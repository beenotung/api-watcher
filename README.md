# api-watcher

Track API usage patterns and changes over time.

Use cases:

- Monitor API credit usage and reset cycles
- Track new version releases
- Detect how often APIs update

Features:

- Schedule-based polling with configurable intervals
- Version tracking based on response body
- JSON path extraction for targeted comparison
- Version history log with duration
